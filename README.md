# Emergentia

Artificial life laboratory running entirely in your browser.

Emergentia is not a chatbot, a game, or a conversational AI. It is a digital laboratory where simple survival rules can produce adaptation, extinction, species divergence, and social behavior over time.

## MVP

The current version focuses on the first observable loop:

- Canvas 2D world renderer
- 50 initial agents
- Food growth and consumption
- Energy, aging, death, reproduction, and mutation
- DNA traits that influence movement, vision, fertility, curiosity, and social behavior
- Species detection through accumulated genetic divergence
- Live dashboard and timeline
- Play, pause, speed, and reset controls

## Stack

- React
- TypeScript
- Vite
- Canvas 2D
